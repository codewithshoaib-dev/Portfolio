# Personal Portfolio

A minimal, client-focused portfolio website built to showcase my work and help SaaS businesses understand what I offer and how to get started.

## Overview

The goal is to clearly communicate services, build trust, and guide potential clients toward getting in touch.

The design focuses on:
- Clear messaging (non-technical, outcome-driven)
- Clean, minimal UI
- Strong visual hierarchy
- Fast, responsive performance

## Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS (with custom component classes)

## Features

- Responsive layout (mobile-first)
- Animated mobile navigation
- Project showcase with focused, client-friendly descriptions
- Simple, direct contact section
- Reusable UI components via Tailwind `@layer components`

## Design Principles

- Minimal and structured (no unnecessary visual noise)
- Neutral color system
- Subtle, intentional animations 
- Content-first approach (copy clarity over decoration)
